## 什么是HTML DOM
HTML DOM 是 HTML 的标准对象模型和编程接口。它定义了：
* 作为对象的 HTML 元素
* 所有 HTML 元素的属性
* 访问所有 HTML 元素的方法
* 所有 HTML 元素的事件
换言之：HTML DOM 是关于如何获取、更改、添加或删除 HTML 元素的标准。
属性为值，方法为操作
